# Тест по Найму - Проект Node.js

Этот проект Node.js был разработан с использованием следующих версий:
- Node.js v18.14.0
- npm v9.3.1

## Запуск Приложения

Для запуска приложения используйте команду:

node server.js/ nodemon server.js

Приложение будет доступно по адресу http://localhost:PORT = 8080

## Функциональности

Этот проект предлагает функциональности управления файлами, такие как загрузка, список, удаление, обновление и скачивание файлов. 
Файлы хранятся в директории "files". 
На текущий момент все авторизованные пользователи могут видеть, изменять и удалять все файлы.
